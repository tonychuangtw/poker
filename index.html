    ICM 結果

    Final Table 分錢計算器

    ICM 分法：依上方「獎金結構」的比例算各玩家 equity，乘上剩餘獎池再加回已鎖定獎金。 Chip-chop：直接依籌碼比例分。剩餘獎池 = 已鎖定獎金之外、尚待分配的金額。玩家 2–9 位。

    推 / 棄決策（Push/Fold $EV）

    使用上方「獎金結構」與「玩家籌碼」（籌碼填本手開始前、未扣盲注的總量）。 模型：hero 全下，只考慮一位指定跟注者（其他人假設全蓋）； range 排名依對隨機手牌勝率；蓋牌時其他玩家籌碼視為不變；平手已按勝率折半計入。

    Nash 均衡圖表（HU push/fold）

    單挑（SB 只能全下或蓋牌、BB 只能跟注或蓋牌）的 chip-EV Nash 均衡， 以 fictitious play 即時求解。綠色 = 全下 / 跟注，灰色 = 蓋牌。

    開牌 range（RFI）

    100bb 現金局／深籌碼 MTT 的標準 raise-first-in 建議 range（無人入池時的開牌手牌）。 按「✏️ 編輯」後點格子可自訂開牌與否。

    已自訂

    面對開牌（3-bet / 防守）

    100bb，前面有人開牌時的應對 range（簡化 GTO 近似）。 紅 = 3-bet、綠 = 跟注、灰 = 蓋牌。 按「✏️ 編輯」後點格子循環切換 棄牌 → 跟注 → 3-bet。

    已自訂

    Outs / 賠率速查

    訓練測驗

    Push/Fold：隨機籌碼深度與手牌，依 Nash 均衡判斷推或棄。 開牌 RFI：隨機位置與手牌，依 6-max 標準開牌 range 判斷加注或蓋牌。 面對開牌：隨機防守情境，判斷 3-bet、跟注或蓋牌。

    📌 測驗以建議 range 為準，圖表的自訂編輯不影響評分。
